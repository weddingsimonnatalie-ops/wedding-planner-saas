Dashboard Presets: Design

4 selectable layouts, one codebase — sections are modular components, presets define ordering

Architecture: Section-based, not layout-based

Each dashboard section (Countdown, Guest Summary, Budget Overview, Payments, etc.) is a standalone component. A preset is just an ordered list of section IDs. This means:

  • Each section coded once, shared across all presets
  • Adding a new section automatically available to all presets
  • Persisted per-user in the database (User.dashboardLayout)
  • Picker in Settings → General, with live preview

The 4 Presets

1. Classic (current default)
Quick Stats (4-col) → Guest Summary + Budget Overview → Meal Choices + Suppliers → Budget by Category → Upcoming Payments → Appointments + Tasks
2. Actions First
Upcoming Payments → Tasks + Appointments → Budget Overview + Budget by Category → Quick Stats (4-col) → Guest Summary + Meals + Suppliers
3. Budget Focus
Countdown + Budget Overview (2-col) → Upcoming Payments + Budget by Category → Tasks + Appointments → Quick Stats (3-col: Guests/Seated/%Paid) → Guest Summary + Meals + Suppliers
4. Organized
AT A GLANCE — Quick Stats (4-col)
NEEDS ATTENTION — Payments + Tasks + Appointments
PROGRESS — Budget + Suppliers + Guest Summary + Meals

Data model change

Add dashboardLayout field to User model (string, default "classic"). Values: classic, actions-first, budget-focus, organized.

Settings picker

In Settings → General, a dropdown or radio group to select layout. Each option has a name and short description. The dashboard re-renders immediately on change (no page reload needed — just update state).

Section components

Extract from DashboardClient into standalone components:

  • DashboardQuickStats — Countdown + Guests + Seated + Budget% (adapts: 4-col or 3-col)
  • DashboardGuestSummary — Guest RSVP breakdown bars
  • DashboardBudgetOverview — Contracted/Paid/Remaining
  • DashboardBudgetCategories — Category progress bars
  • DashboardMeals — Meal choice bars
  • DashboardSuppliers — Status pill list
  • DashboardPayments — Overdue/upcoming payments list
  • DashboardAppointments — Upcoming appointments list
  • DashboardTasks — Overdue/due-soon tasks list
